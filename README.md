# 纳西妲的 VCP 插件源

围绕元思考链系统设计的 4 个 VCP 插件：增强、观察、自检、Lisp。

## 在 AdminPanel 添加这个源

1. 打开 AdminPanel → 插件商店 → 源管理
2. 类型：GitHub 仓库
3. URL：https://github.com/390538316/nahida-vcp-plugins
4. 切到插件市场 → 刷新 → 4 个插件出现 → 点击安装

## 包含的插件

- 元思考链增强器 (MetaThinkingEnhancer)
- 元思考链观察器 (MetaThinkingObserver)
- VCP 系统自检器 (VCPInspector)
- 纳西妲 Lisp 解释器 (NahidaLisp)
